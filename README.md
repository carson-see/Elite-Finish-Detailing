# Elite Finish Detailing

Static website for **Elite Finish Detailing** — a mobile car detailing service based in Shelby, NE, serving the greater Omaha metro.

**Owners:** Nixon Thelen · (402) 679-7198 · and Blake Schmidt · (402) 278-4466
**Tagline:** Drive Clean. Drive Confident.

## Stack
- Single-file static HTML (`index.html`)
- Tailwind CSS via CDN
- No build step, no backend, no database
- Deployed to GitHub Pages

## Local preview
Open `index.html` directly in a browser. That's it.

## Editing
See [`HANDOFF.md`](./HANDOFF.md) for the full owner's manual: setting up Formspree for bookings, Stripe for payments, Google Business Profile, custom domain, and swapping in real photos.
