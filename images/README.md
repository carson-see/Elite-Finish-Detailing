# Images folder

Drop the following files here and the site will pick them up automatically:

## Required
- **`logo-wordmark.png`** — the gold car silhouette + "ELITE FINISH DETAILING" wordmark.
  Used in: about section hero image.
- **`logo-favicon.png`** — the gold "EF" monogram on black.
  Used in: browser tab favicon + top-nav logo + footer logo.

## Recommended (for the Before/After gallery)
- `before-1.jpg` / `after-1.jpg` — first customer car (e.g. interior)
- `before-2.jpg` / `after-2.jpg` — second customer car (e.g. exterior)
- `before-3.jpg` / `after-3.jpg` — optional third pair

## Hero
- `hero.jpg` — optional large background photo for the top of the page (1920px wide+)
  If this file doesn't exist, the site falls back to a stock image.

Filenames are case-sensitive on Linux servers (including GitHub Pages). Keep them lowercase.
